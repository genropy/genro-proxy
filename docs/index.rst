genro-proxy Documentation
=========================

**genro-proxy** is a base infrastructure package for building Genro microservices and proxies.

It provides multi-tenant architecture, automatic API/CLI generation, and database abstraction
as a foundation for domain-specific services like mail proxies, storage proxies, or API gateways.
